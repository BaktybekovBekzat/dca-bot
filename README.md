# dca-bot

Инструкция по запуску проекта

### Установка зависимостей

```sh
npm install
```

### Запуск backend сервера

```sh
npm run dev:server
```

### Локальный запуск приложения

```sh
npm run dev
```

### Компиляция production-сборки

```sh
npm run build
```

### Запуск production-сборки

```sh
npm run preview
```
